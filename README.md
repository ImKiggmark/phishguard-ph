# PhishGuard PH

PhishGuard PH is a cybersecurity reporting web application that allows users to report phishing attacks and suspicious links. The system automatically classifies reports into risk levels and provides an admin dashboard for monitoring threats.

## Features

- User Registration and Login
- Phishing Report Submission
- Risk Detection (Low, Medium, High)
- View and Manage Reports
- Admin Dashboard
- Threat Intelligence Feed
- CRUD Operations
- Navigation System

## Technologies Used

- PHP
- MySQL
- XAMPP
- HTML / CSS
- GitHub
